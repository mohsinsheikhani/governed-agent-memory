# Bucket 1 — Clean / happy path

Control group. Deals progress believably and the agent should find nothing alarming.
No contradictions, no consequential hedges, no poisoning. If a "clean" stream hides a
flip or a hedge it pollutes the baseline, so each one is kept honest.
